# BetterSwordFight
Chivalry swordfight in minecraft!

# Workspace Configuration

```
./gradlew setupDecompWorkspace
```

For eclipse users:

```
./gradlew eclipse
```

And import BetterSwordFight as a gradle project

# Run

In Eclipse:

Run > Run configurations > BetterSwordFight_Client
